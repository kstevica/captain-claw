# Brief Format

Use this output shape:

```markdown
# <Title>

## Summary
- 3-6 bullets with the most important conclusions.

## Findings
### 1) <Finding title>
- Evidence:
  - <claim> ([source](https://...))
  - <claim> ([source](https://...))
- Why it matters:
  - <short impact statement>

### 2) <Finding title>
- Evidence:
  - <claim> ([source](https://...))
- Why it matters:
  - <short impact statement>

## Conflicts / Uncertainty
- <where sources disagree or where confidence is low>

## Sources
1. https://...
2. https://...
```
